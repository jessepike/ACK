# AI Dev Project Template — Lean

Minimal, production-oriented AI/software dev starter.
No notebooks. No local models. No bloat.

Stack:
- Python (venv + requirements.txt)
- Node.js / TypeScript
- .env-based secrets
- Homebrew for system deps

Quick start:
brew bundle
make python
make node
